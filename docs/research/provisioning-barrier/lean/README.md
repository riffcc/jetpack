# Lean 4 model + proofs

Layer-by-layer formalization of the provisioning-barrier protocol. L0 (core state
machine + strict-all barrier) first; L1 (%-threshold), L2 (max-failures), L3
(play-module composition) added incrementally. Driven via the riff-lean-prover
skill; see `../README.md` for the layer plan and properties.
